# FreeCodeCamp Python Projects

## Projects

**build_rpg_character.py** - RPG character creator with stat validation (STR, INT, CHA must sum to 7, range 1-4 each)

**caesar_cipher.py** - Caesar cipher encryption/decryption with shift values 1-25

**config_manager.py** - User configuration manager for settings (theme, language, notifications) with add/update/delete/view functions

**for_else.py** - Demonstrates Python's for-else loop structure to find words without vowels

**match.py** - HTTP error code handler using Python's match-case statement for status codes 400-503

**number_generator.py** - Number pattern generator that creates sequences from 1 to n with input validation

## Run

```bash
python build_rpg_character.py
python caesar_cipher.py
python config_manager.py
python for_else.py
python match.py
python number_generator.py
```

# FreeCodeCamp Python Projects

## Projects

**build_rpg_character.py** - RPG character creator with stat validation (STR, INT, CHA must sum to 7, range 1-4 each)

**caesar_cipher.py** - Caesar cipher encryption/decryption with shift values 1-25

## Run

```bash
python build_rpg_character.py
python caesar_cipher.py
```
